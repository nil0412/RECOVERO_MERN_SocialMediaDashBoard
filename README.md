## Run Locally

Clone the project

```bash
  git clone https://github.com/nil0412/RECOVERO_MERN_SocialMediaDashBoard.git
```

Go to the project directory

```bash
  cd RECOVERO_MERN_SocialMediaDashBoard
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
